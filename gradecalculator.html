Assembly Grade Calculator


The purpose of this project was to determine if students passed a class. This was coded in a distribution of assembly language using Index Register X and Y and Accumulator A and B. It was loaded onto a Wytec Dragon12 Plus board and would display a P or F on parallel ports. https://github.com/KMDean/Assembly-Grade-Calculator

The grades are out of 10 each. The first three grades are Practical Grades and the last five grades are Theory Grades. In order to pass, a student must have over 50% in their Practical Grades and over 50% in their Theory Grades.

The code works using subroutines such as Calculate_Average which calculates the average of the students' grades, Pass_Fail which determines if the students passed or failed, Config_HEX_Displays which configured the board hardware's 7-segment Hex Displays, and PF_HEX_Display which determines whether to show P or F and adds a delay between each letter display.

Pass_Fail_Calc is the main that contains constants and three loops.
Grades were given as:

; Student A
db 10,4,12
db 10,10,10,4,9
; Student B
db 7,8,20
db 3,0,4,4,3
