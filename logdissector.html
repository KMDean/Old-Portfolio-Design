Wireshark Log Dissector in TShark


The network project was to dissect information from a wireshark capture. This includes conversation and flow statistics, displaying the packets, and the amount of frames in certain filters. This was written in batch and in TShark. https://github.com/KMDean/Wireshark-Log-Dissector

This is how the batch file displays the amount of frames depending on the filter in a table format and allows the user to enter another wireshark capture to dissect.
