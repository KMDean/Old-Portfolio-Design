Subnet Calculator GUI in Java


The network project was to create a subnet calculator in our preferred language. Originally, I created it as a console based program but I recently created a quick GUI to get extra practice and warm up for my second year of Computer Engineering.

This short clip demonstrates how the program works. It takes in an IP Address and Subnet Mask then, returns the information on that exact subnet and allows you to display all possible subnets in a table format.
